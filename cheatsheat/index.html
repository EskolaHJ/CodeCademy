HTML and CSS Cheat Sheet

Tag/Property Example Description
HTML Tags
<a> <a href="https://www.example.com">Link</a> Defines a hyperlink.
<img> <img src="image.jpg" alt="Description"> Embeds an image in the document.
<div> <div>Content</div> Defines a division or a section in an HTML document.
<span> <span>Text</span> Defines a section in a document.
<table> <table><tr><td>Data</td></tr></table> Defines a table.
<h1> - <h6> <h1>Heading 1</h1> Defines HTML headings from level 1 to 6.
<p> <p>Paragraph</p> Defines a paragraph.
<ul> <ul><li>Item</li></ul> Defines an unordered list.
<ol> <ol><li>Item</li></ol> Defines an ordered list.
<li> <li>Item</li> Defines a list item.
<form> <form>Form content</form> Defines an HTML form for user input.
<input> <input type="text"> Defines an input control.
<button> <button>Click me</button> Defines a clickable button.
<nav> <nav>Navigation</nav> Defines navigation links.
<header> <header>Header content</header> Defines a header for a document or section.
<footer> <footer>Footer content</footer> Defines a footer for a document or section.
CSS Properties
color color: red; Sets the color of the text.
background-color background-color: blue; Sets the background color of an element.
font-size font-size: 16px; Sets the size of the font.
font-family font-family: Arial, sans-serif; Sets the font family of the text.
margin margin: 10px; Sets the margin around an element.
padding padding: 10px; Sets the padding inside an element.
border border: 1px solid black; Sets the border around an element.
width width: 100px; Sets the width of an element.
height height: 100px; Sets the height of an element.
text-align text-align: center; Sets the horizontal alignment of text.
display display: block; Sets the display behavior of an element.
position position: absolute; Sets the positioning method for an element.
top, right, bottom, left top: 10px; left: 10px; Sets the position of an absolutely positioned element.
