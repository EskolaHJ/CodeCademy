HTML, CSS, and JavaScript Cheat Sheet

HTML Tags

Tag/Property Example Description
<a> <a href="https://www.example.com">Link</a> Defines a hyperlink.
<img> <img src="image.jpg" alt="Description"> Embeds an image in the document.
<div> <div>Content</div> Defines a division or a section in an HTML document.
<span> <span>Text</span> Defines a section in a document.
<table> <table><tr><td>Data</td></tr></table> Defines a table.
<h1> - <h6> <h1>Heading 1</h1> Defines HTML headings from level 1 to 6.
<p> <p>Paragraph</p> Defines a paragraph.
<ul> <ul><li>Item</li></ul> Defines an unordered list.
<ol> <ol><li>Item</li></ol> Defines an ordered list.
<li> <li>Item</li> Defines a list item.
<form> <form>Form content</form> Defines an HTML form for user input.
<input> <input type="text"> Defines an input control.
<button> <button>Click me</button> Defines a clickable button.
<nav> <nav>Navigation</nav> Defines navigation links.
<header> <header>Header content</header> Defines a header for a document or section.
<footer> <footer>Footer content</footer> Defines a footer for a document or section.
<section> <section>Section content</section> Defines a section in a document.
<article> <article>Article content</article> Defines an article.
<aside> <aside>Aside content</aside> Defines content aside from the main content.
<main> <main>Main content</main> Defines the main content of a document.
<figure> <figure><img src="image.jpg" alt="Description"><figcaption>Caption</figcaption></figure> Groups media content with a caption.
<figcaption> <figcaption>Caption</figcaption> Defines a caption for a figure element.
<code> <code>Code snippet</code> Defines a piece of computer code.
<pre> <pre>Preformatted text</pre> Defines preformatted text.

CSS Properties

Tag/Property Example Description
color color: red; Sets the color of the text.
background-color background-color: blue; Sets the background color of an element.
font-size font-size: 16px; Sets the size of the font.
font-family font-family: Arial, sans-serif; Sets the font family of the text.
margin margin: 10px; Sets the margin around an element.
padding padding: 10px; Sets the padding inside an element.
border border: 1px solid black; Sets the border around an element.
width width: 100px; Sets the width of an element.
height height: 100px; Sets the height of an element.
text-align text-align: center; Sets the horizontal alignment of text.
display display: block; Sets the display behavior of an element.
position position: absolute; Sets the positioning method for an element.
top, right, bottom, left top: 10px; left: 10px; Sets the position of an absolutely positioned element.
float float: left; Specifies how an element should float.
clear clear: both; Specifies what elements can float beside the cleared element.
overflow overflow: hidden; Specifies what happens if content overflows an element's box.
visibility visibility: hidden; Specifies whether an element is visible or hidden.
opacity opacity: 0.5; Sets the opacity level of an element.
z-index z-index: 10; Sets the stack order of an element.
box-shadow box-shadow: 0px 4px 2px -2px gray; Adds shadow effects around an element's frame.
border-radius border-radius: 10px; Defines the radius of the element's corners.
background-image background-image: url('image.jpg'); Sets the background image for an element.
background-size background-size: cover; Specifies the size of the background image.
transition transition: all 0.3s ease; Defines the transition effect between two states of an element.
transform transform: rotate(45deg); Applies a 2D or 3D transformation to an element.
flex flex: 1; Specifies how a flex item will grow relative to the rest of the flex items.
flex-direction flex-direction: row; Defines the direction of the flexible items.
justify-content justify-content: center; Aligns the flexible container's items when the items do not use all available space.
align-items align-items: center; Aligns flex items along the cross axis.

JavaScript Basics

Tag/Property Example Description
Variables let x = 10; // Declares a block-scoped variable Declares a block-scoped variable.
Constants const PI = 3.14; // Declares a block-scoped, read-only constant Declares a block-scoped, read-only constant.
Functions function myFunction() { /* code */ } // Declares a function Declares a function.
Arrow Functions const myFunc = () => { /* code */ } // Declares an arrow function expression Declares an arrow function expression.
Event Listeners element.addEventListener('click', function) // Adds an event listener to an element Adds an event listener to an element.
Arrays let arr = [1, 2, 3]; // Declares an array Declares an array.
Objects let obj = {key: 'value'}; // Declares an object Declares an object.
Loops for (let i = 0; i < 5; i++) { /* code */ } // A for loop Creates a loop that repeats code a specified number of times.
Conditionals if (x > 10) { /* code */ } // An if statement Executes code if a specified condition is true.
DOM Manipulation document.getElementById('id').textContent = 'Hello'; // Changes the text content of an element Accesses and manipulates HTML elements.
Console Log console.log('Hello World'); // Logs a message to the console Prints a message to the web console.
